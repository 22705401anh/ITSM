?? NOTIFICATION TYPES

SUCCESS (Green)
  - Check mark icon animates in
  - Used for: License created, Asset added
  - Auto-closes after OK click
  - Optional redirect

ERROR (Red)
  - Exclamation mark shakes
  - Used for: Failed operations, Errors
  - Stays until user clicks OK

WARNING (Orange)
  - Warning triangle pulses
  - Used for: Confirmations needed

INFO (Blue)
  - Info circle bounces
  - Used for: Information messages

?? TECHNICAL DETAILS

Files Involved:
  ? ITSM/app/web/static/js/notifications.js
     - Contains all notification logic
     - Creates professional modals
     - Handles animations
     - Available as global `Notification` object

  ? ITSM/app/web/templates/base.html
     - Loads notifications.js on every page
     - Makes `Notification` globally available

  ? ITSM/app/web/templates/licenses/list.html
     - Uses Notification.success() for success
     - Uses Notification.error() for errors

  ? ITSM/app/web/templates/assets/create.html
     - Uses Notification.success() for success
     - Uses Notification.error() for errors
     - Redirects after success
