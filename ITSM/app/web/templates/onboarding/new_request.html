Employee Details
Manager Assignment
The assigned manager will receive a task to select the required hardware and software. Searches Active Directory.
Workflow Process

Step 1: HR Request
Submit the new hire's basic information.

Step 2: Manager Approval
The assigned manager will be notified to select required hardware and software.

Step 3: IT Provisioning
IT receives automated tasks to create accounts and provision selected equipment.

Important Guidelines
  • Ensure names match official identification.
  • Submit requests at least 5 days prior to the start date to ensure IT SLAs are met.
  • Correctly specifying the Direct Manager is crucial for avoiding approval delays.
