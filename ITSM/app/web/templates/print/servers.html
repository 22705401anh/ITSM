KOSTAL Print Agent Setup
The KOSTAL Print Agent runs as a background service on your Windows Print Servers to monitor print jobs, page counts, and queue health in real-time.
Step-by-Step Installation
  1. Copy Agent Files
    Copy the scripts/print_agent/ directory from the ITSM server to your target Windows Print Server.
  2. Install Python
    Ensure Python 3.10+ is installed on the Print Server and added to the System PATH.
  3. Install Dependencies
    Open an Administrator Command Prompt, navigate to the agent folder, and run:
    pip install -r requirements.txt
  4. Configure the Agent
    Rename config.json.example to config.json and set the ITSM server address:
    {
  "server_url": "http://YOUR_ITSM_IP:8000",
  "log_level": "INFO"
}
  5. Install as Windows Service
    Run the provided script as Administrator to install and start the background service:
    install_service.bat
Once started, the agent will automatically appear in the server list below.
