Agent Installation Guide
1. Deploy Agent Files

Copy the scripts/print_agent/ folder to the target Windows print server:

# Copy to print server
xcopy /E /I scripts\print_agent C:\KOSTAL\PrintAgent
2. Configure

Copy config.json.example to config.json and set the ITSM server URL:

{
  "server_url": "http://<ITSM_HOST>:8000",
  "poll_interval_seconds": 10,
  "batch_size": 50,
  "log_level": "INFO"
}
3. Install & Start
# Install Python dependencies
pip install -r requirements.txt
# Install Windows Service
install_service.bat
# Or run interactively for testing
python kostal_print_agent.py --console

Security
  • Agent authenticates via X-Agent-Token header
  • Token is SHA-256 hashed in the database
  • Web UI requires JWT + print permission
  • All changes are written to the Audit Log
Offline Resilience
  • Agent caches jobs locally in SQLite when offline
  • Automatic batch sync when connection is restored
  • SHA-256 correlation IDs prevent duplicate submissions
