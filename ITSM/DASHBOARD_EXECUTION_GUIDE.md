?? TROUBLESHOOTING

Issue: Dashboard doesn't load
Solution:
  1. Check application is running (check terminal)
  2. Try refreshing page (Ctrl+R or F5)
  3. Check URL is correct: http://localhost:8000/dashboard
  4. Clear browser cache

Issue: No data shows (all zeros)
Solution:
  1. Make sure you have added some assets/licenses first
  2. Check API endpoints in browser console (F12)
  3. Add test data in /assets or /licenses
  4. Refresh page

Issue: Styling looks broken
Solution:
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Hard refresh (Ctrl+Shift+R)
  3. Try different browser
  4. Check Bootstrap CSS is loaded

Issue: Buttons don't work
Solution:
  1. Check console for errors (F12)
  2. Verify API endpoints are accessible
  3. Check network tab in DevTools
  4. Verify application is still running

Issue: Mobile view is broken
Solution:
  1. Check responsive breakpoints
  2. Test in DevTools mobile view
  3. Try actual mobile device
  4. Check viewport meta tag in base.html
