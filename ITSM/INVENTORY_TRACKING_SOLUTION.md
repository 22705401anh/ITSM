?? API ENDPOINTS

ASSETS API
????????????????????????????????????????
GET     /api/assets                    - List all assets
POST    /api/assets                    - Create new asset
GET     /api/assets/{asset_id}         - Get asset details
PUT     /api/assets/{asset_id}         - Update asset
DELETE  /api/assets/{asset_id}         - Delete asset
POST    /api/assets/{asset_id}/maintenance - Record maintenance
GET     /api/assets/{asset_id}/maintenance - Get maintenance history

LICENSES API
????????????????????????????????????????
GET     /api/licenses                  - List all licenses
POST    /api/licenses                  - Create new license
GET     /api/licenses/{license_id}     - Get license details
PUT     /api/licenses/{license_id}     - Update license
DELETE  /api/licenses/{license_id}     - Delete license
GET     /api/licenses/{license_id}/access-history - View access logs

DOCUMENTATION API
????????????????????????????????????????

DELETE  /api/documentation/{doc_id}    - Delete documentation
