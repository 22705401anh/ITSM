?? NOTIFICATION PARAMETERS

show(type, title, message, callback)
  type: 'success' | 'error' | 'warning' | 'info'
  title: String (e.g., "License Created!")
  message: String (e.g., "Your license has been registered.")
  callback: Function | null (optional, called after modal closes)

Shortcuts:
  success(title, message, callback)
  error(title, message, callback)
  warning(title, message, callback)
  info(title, message, callback)

?? HOW IT WORKS

1. User submits form (License or Asset)
2. Form data is sent via fetch API
3. Server processes the request
4. Response received (success or error)
5. Professional modal pops up with result
6. Modal shows colored gradient header with icon
7. Message displayed with full context
8. User clicks OK button to dismiss
9. Optional callback function executes
10. Redirect or refresh happens
