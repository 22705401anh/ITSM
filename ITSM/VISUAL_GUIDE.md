## ? User Experience Flow

### First Time User
```
1. Visit page
2. See full sidebar (default)
3. Scroll down to Administration
4. See new "AD Users" link
5. Can click to view AD users
6. Can toggle sidebar to explore
7. Preference saved
```

### Returning User
```
1. Visit page
2. Sidebar shows saved state
3. If previously collapsed ? shows collapsed
4. If previously expanded ? shows expanded
5. Can toggle anytime
6. New preference saved
```

### Mobile User
```
1. Visit on phone
2. Sidebar takes full width
3. Tap hamburger to collapse
4. More screen space
5. Tap again to expand
6. State saved
```
