?? DATA LOADING & DISPLAY

The dashboard loads data from these APIs:

Assets:
  GET /api/assets/stats/summary
  - Returns: total_assets, by_status, by_type, total_licenses
  - Displays: Total count, status breakdown, type list

Licenses:
  GET /api/licenses
  - Returns: Array of license objects
  - Displays: Total, active, expired, expiring counts

Problems:
  GET /api/problems
  - Returns: Array of problem objects
  - Displays: Open problems count

Changes:
  GET /api/changes
  - Returns: Array of change objects
  - Displays: Pending changes count

All with error handling and fallback values!
