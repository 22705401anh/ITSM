## Technical Changes

### CSS Updates
```css
/* Old (problematic) */
.sidebar {
    width: 100%;  /* ? Full page! */
}

/* New (fixed) */
.sidebar {
    width: 250px;  /* ? Normal size */
    flex-shrink: 0;  /* ? Doesn't shrink */
}
```

### HTML Layout
```html
<!-- Old (Bootstrap grid) -->
<div class="row g-0">
    <div class="col-md-3">Sidebar</div>
    <div class="col-md-9">Content</div>
</div>

<!-- New (Flexbox) -->
<div class="d-flex">
    <aside class="sidebar">Sidebar</aside>
    <main class="flex-grow-1">Content</main>
</div>
```
